###  docker 服务

http代理  隐藏和伪装对外web服务 提高安全性

````bash
docker run -d --name proxym --restart=always --net=host -v /opt/proxym/data:/data -v /opt/proxym/encrypt:/etc/letsencrypt   chishin/nginx-proxy-manager-zh #jc21/nginx-proxy-manager

````
