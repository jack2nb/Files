#### 状态

State = 与对端的状态/对端的角色   

状态都必须full状态才正常  ，ExStart状态为理论主路由被占用

| 状态                 | 步骤        | 备注               |
| -------------------- | ----------- | ------------------ |
| **· Down状态；**     | 离线        |                    |
| **· Attempt状态；**  | 单向联络    |                    |
| **· Init状态；**     | 收到hello   |                    |
| **· 2-way状态；**    | 得到对方rid | vm主机不能ospf服务 |
| **· Exstart状态；**  | 选举        | vm主机不能ospf服务 |
| **· Exchange状态；** | 开始交换    |                    |
| **· Loading状态；**  | 交互路由表  |                    |
| **· Full状态。**     | 完成交换    |                    |



####  角色

DR指定路由，BDR备选路由，DROther其他路由

显示的远端路由器扮演的角色
